# Vrinda Store Sales Analysis Dashboard

## Project Overview
- This project presents an interactive sales dashboard built using Microsoft Excel. 
- It analyzes sales performance, customer demographics, and order trends for Vrinda Store.
- The dashboard helps in understanding key business insights such as top-performing states, gender-based purchasing behavior, and monthly sales trends.

## Dataset used
- <a href="https://github.com/Jayesh-Gurav/Sales-Analysis-Excel-Dashboard/blob/main/Vrinda%20Store%20Data%20Analysis.xlsx">Dataset</a>

## Objectives
- Analyze overall sales performance
- Identify top-performing states
- Understand customer demographics (age & gender)
- Track monthly sales and order trends
- Evaluate order status distribution (Delivered, Cancelled, Returned)

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

## Dashboard Features
- Top 5 States by Sales
- Age Category vs Gender Analysis
- Monthly Sales vs Orders Trend
- Sales Distribution by Status
- Gender-wise Sales Contribution
- Interactive Filters (Month, State, Category, Status)

## Key Insights
- Women contribute around 64% of total sales
- Majority of orders are successfully delivered (~93%)
- Maharashtra and Karnataka are top-performing states
- Adult age group has the highest purchasing contribution
- Sales show fluctuations across months with peak periods
